🌐 Publicar en web con GitHub Pages — Guía paso a paso

Gratis, sin servidor, sin técnicos. La gente verá el ranking con tu URL personalizada.

1
Crea una cuenta en GitHub
Ve a github.com y regístrate gratis con tu email. Es el paso más sencillo.
2
Crea un repositorio nuevo
En GitHub, pulsa el botón verde "New". Ponle el nombre: ranking-evo-amf. Marca la casilla "Add a README file". Pulsa "Create repository".
3
Sube el archivo HTML
Dentro del repositorio, pulsa "Add file" → "Upload files". Sube el archivo ranking-evo-amf.html y renómbralo como index.html antes de subirlo. Pulsa "Commit changes".
4
Activa GitHub Pages
Ve a Settings → Pages (menú lateral). En "Source" selecciona "Deploy from a branch". En "Branch" elige "main" y carpeta "/ (root)". Pulsa "Save".
5
Tu web ya está online
En 1-2 minutos tu URL será:https://TU-USUARIO.github.io/ranking-evo-amf/Comparte esa URL con todos. Para actualizar datos: exporta el JSON desde aquí, cárgalo en la versión local, y vuelve a subir el index.html a GitHub.
💡
Consejo: versión pública sin contraseña
La web pública debería tener la contraseña desactivada para que todos puedan ver el ranking. Cuando la genere para ti, incluirá un botón "Generar versión pública" que crea un HTML sin login para subir a GitHub.
